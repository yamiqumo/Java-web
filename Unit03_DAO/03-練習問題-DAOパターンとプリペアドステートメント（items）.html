第3回：DAOパターンの練習問題（items）〜プリペアドステートメントの練習〜

この演習の目的

DAOに JDBC を集約し、PreparedStatement の ? と setXxx の対応を体に入れる練習です。

  • 題材は items（ECサイトの商品）
  • SQL文字列に値を連結しない（必ず ? に入れてバインド）
  • try-with-resources でリソースを適切に閉じる

基本問題（3問・items）

目標：DAOクラス内で PreparedStatement を使い、ResultSet を読み取り処理する

問題1：カテゴリで絞り込み — category 指定で商品名と単価を表示せよ。

ヒント

SQL に WHERE category = ? を使い、ps.setString(1, category) で値を渡します。

解答例 
public class ItemDao {

	public void printItemsByCategory(String category) throws SQLException {
		String sql = "SELECT item_name, unit_price FROM items WHERE category = ?";

		try (Connection con = DriverManager.getConnection(URL, USER, PASS);
		     PreparedStatement ps = con.prepareStatement(sql)) {

			ps.setString(1, category);

			try (ResultSet rs = ps.executeQuery()) {
				while (rs.next()) {
					String name = rs.getString("item_name");
					int price   = rs.getInt("unit_price");
					System.out.println(name + " : " + price + "円");
				}
			}
		}
	}
}

問題3：合計金額 — items の unit_price の合計を DAO で計算し、Java 側で表示せよ。

ヒント

SQL で全件を SELECT unit_price FROM items して、ループで total += rs.getInt("unit_price") の形で合計します。

解答例 
public class ItemDao {

	public int sumUnitPrices() throws SQLException {
		String sql = "SELECT unit_price FROM items";
		int total = 0;

		try (Connection con = DriverManager.getConnection(URL, USER, PASS);
		     PreparedStatement ps = con.prepareStatement(sql);
		     ResultSet rs = ps.executeQuery()) {

			while (rs.next()) {
				total += rs.getInt("unit_price");
			}
		}
		return total;
	}
}
