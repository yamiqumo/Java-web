①【問題】参照系（基礎）— findById

やること：ItemDao に findById(int itemId) を実装する。次の ??? を埋めよ。

  • SQL の ???：SELECT する列名（4列）
  • ???：PreparedStatement に渡す SQL 文字列全体
  • ???：ps.set??? で主キーをバインド（番号は 1）
  • ???：ItemDto のコンストラクタに渡す rs.get??? の組み合わせ
ヒント

WHERE 句は item_id = ?。1件だけなら if (rs.next()) で十分。

解答例 
public ItemDto findById(int itemId) throws SQLException {
	String sql = "SELECT item_id, category, item_name, unit_price FROM items WHERE item_id = ?";

	try (Connection con = getConnection();
			PreparedStatement ps = con.prepareStatement(sql)) {

		ps.setInt(1, itemId);

		try (ResultSet rs = ps.executeQuery()) {
			if (rs.next()) {
				return new ItemDto(
						rs.getInt("item_id"),
						rs.getString("category"),
						rs.getString("item_name"),
						rs.getInt("unit_price"));
			}
			return null;
		}
	}
}

②【問題】更新系（基礎）— insert

やること：insert(ItemDto item) を実装する。次の ??? を埋めよ。

  • ???：INSERT 文の列と VALUES の ? の個数（4つ）
  • ???：ps.setInt / setString の順番（1〜4番目）
  • ???：更新系の実行メソッド（戻り値は int）
ヒント

INSERT の実行は executeUpdate()。SELECT とは違う。

解答例 
public int insert(ItemDto item) throws SQLException {
	String sql = "INSERT INTO items (item_id, category, item_name, unit_price) VALUES (?, ?, ?, ?)";

	try (Connection con = getConnection();
			PreparedStatement ps = con.prepareStatement(sql)) {

		ps.setInt(1, item.getItemId());
		ps.setString(2, item.getCategory());
		ps.setString(3, item.getItemName());
		ps.setInt(4, item.getUnitPrice());

		return ps.executeUpdate();
	}
}

③【問題】参照系（発展）— 条件＋並び替え

やること：findByCategoryAndMinPriceOrderByPrice(String category, int minPrice) を実装する。カテゴリが一致し、かつ単価が minPrice 以上の行を、単価の昇順で返す。

  • ???：WHERE に並べる条件（category と unit_price）
  • ???：ORDER BY 句
  • ???：setString(1, ...) と setInt(2, ...) の対応
ヒント

複数行は while (rs.next()) で ArrayList に追加。

解答例 
public List<ItemDto> findByCategoryAndMinPriceOrderByPrice(String category, int minPrice) throws SQLException {
	String sql = "SELECT item_id, category, item_name, unit_price FROM items "
			+ "WHERE category = ? AND unit_price >= ? ORDER BY unit_price ASC";
	List<ItemDto> list = new ArrayList<>();

	try (Connection con = getConnection();
			PreparedStatement ps = con.prepareStatement(sql)) {

		ps.setString(1, category);
		ps.setInt(2, minPrice);

		try (ResultSet rs = ps.executeQuery()) {
			while (rs.next()) {
				list.add(new ItemDto(
						rs.getInt("item_id"),
						rs.getString("category"),
						rs.getString("item_name"),
						rs.getInt("unit_price")));
			}
		}
	}
	return list;
}

④【問題】更新系（発展）— update（全列）

やること：update(ItemDto item) を実装する。主キー item_id で行を特定し、category / item_name / unit_price を更新する。

  • ???：SET 句と WHERE 句の ? の並び（SET が1〜3、WHERE が4）
  • ???：executeUpdate() の戻り値を返す
ヒント

WHERE の item_id は最後の ? にバインドすると整理しやすい。

解答例 
public int update(ItemDto item) throws SQLException {
	String sql = "UPDATE items SET category = ?, item_name = ?, unit_price = ? WHERE item_id = ?";

	try (Connection con = getConnection();
			PreparedStatement ps = con.prepareStatement(sql)) {

		ps.setString(1, item.getCategory());
		ps.setString(2, item.getItemName());
		ps.setInt(3, item.getUnitPrice());
		ps.setInt(4, item.getItemId());

		return ps.executeUpdate();
	}
}
