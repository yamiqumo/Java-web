基本問題（emps のみ・SELECT）

目標：EmpDao のメソッドを呼び出すか、同じ SQL を PreparedStatement で実行し、結果をコンソールに出す。

問題1：全件表示 — EmpDao#findAll を使い、全員の苗字・名前・給与を1行ずつ表示せよ。

ヒント

main で new EmpDao().findAll() を呼び、返った List<EmpDto> を拡張 for で回す。throws SQLException は main に付けてよい。

解答例 
public static void main(String[] args) throws Exception {
	EmpDao dao = new EmpDao();
	for (EmpDto e : dao.findAll()) {
		System.out.println(e.getFname() + e.getLname() + " " + e.getSalary());
	}
}

問題2：1件取得 — emp_id = 7 の社員を findById で取得し、見つからない場合は null と分かるように表示せよ。

ヒント

if (e == null) の分岐を書く。

解答例 
EmpDto e = new EmpDao().findById(7);
if (e == null) {
	System.out.println("該当なし");
} else {
	System.out.println(e.getFname() + e.getLname() + " " + e.getSalary());
}

問題3：部署で絞り込み — dept_no = 20（開発部）の社員だけを findByDeptNo で取得し、苗字と名前を表示せよ。

ヒント

EmpDao に findByDeptNo がまだ無い場合は、講義テキストの findById をコピーして SQL とバインドだけ変えて実装する。

解答例 
for (EmpDto e : new EmpDao().findByDeptNo(20)) {
	System.out.println(e.getFname() + e.getLname());
}

問題4：Java で集計 — findAll で取得したリストから、給与が 300000 以上の人数を数えて表示せよ（SQL の COUNT は使わず、ループで数える）。

ヒント

カウンタ変数を用意し、getSalary() >= 300000 のときだけ増やす。答えは 4 名。

解答例 
int n = 0;
for (EmpDto e : new EmpDao().findAll()) {
	if (e.getSalary() >= 300000) {
		n++;
	}
}
System.out.println("300000以上: " + n + "名");

発展問題（depts と emps・JOIN）

目標：JOIN で部署名を取り、EmpDto#getDeptName に載せる。

問題5：部署名つき一覧 — EmpDao#findAllWithDeptName を使い、「苗字＋名前＋部署名＋給与」を1行ずつ表示せよ。

ヒント

SQL は emps e JOIN depts d ON e.dept_no = d.dept_no。rs.getString("dept_name") を DTO の第6引数に渡す。

解答例 
for (EmpDto e : new EmpDao().findAllWithDeptName()) {
	System.out.println(
			e.getFname() + e.getLname() + " " + e.getDeptName() + " " + e.getSalary());
}

問題6：メソッド追加 — findByIdWithDeptName(int empId) を EmpDao に新規実装せよ。JOIN し、WHERE e.emp_id = ? で1件に絞る。見つからなければ null。

ヒント

findById と findAllWithDeptName の SQL を組み合わせる。

解答例（メソッド本体のイメージ） 
String sql =
	"SELECT e.emp_id, e.fname, e.lname, e.salary, e.dept_no, d.dept_name "
			+ "FROM emps e JOIN depts d ON e.dept_no = d.dept_no WHERE e.emp_id = ?";
try (Connection con = getConnection();
		PreparedStatement ps = con.prepareStatement(sql)) {
	ps.setInt(1, empId);
	try (ResultSet rs = ps.executeQuery()) {
		if (!rs.next()) {
			return null;
		}
		return new EmpDto(
				rs.getInt("emp_id"),
				rs.getString("fname"),
				rs.getString("lname"),
				rs.getInt("salary"),
				rs.getInt("dept_no"),
				rs.getString("dept_name"));
	}
}

問題7：部署ごとの人数 — findAllWithDeptName の結果だけを使い（SQL で GROUP BY しない）、部署名ごとの人数をコンソールに出力せよ。

ヒント

Map<String, Integer> に部署名をキーにして件数を足す。発展として後日 GROUP BY を DAO に書く形に置き換えてもよい。

解答例 
Map<String, Integer> map = new HashMap<>();
for (EmpDto e : new EmpDao().findAllWithDeptName()) {
	String d = e.getDeptName();
	map.put(d, map.getOrDefault(d, 0) + 1);
}
for (Map.Entry<String, Integer> en : map.entrySet()) {
	System.out.println(en.getKey() + " : " + en.getValue() + "名");
}

import java.util.HashMap; import java.util.Map; が必要。
