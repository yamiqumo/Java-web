---

## 1. 学習目標（項目と説明）

- **SQLとは何か**  
  - DBにお願いする言語
- **SELECT**  
  - 見る（取り出して結果表を作る）働き
- **INSERT**  
  - 1行追加の働き
- **手順の型**  
  - `INSERT ▶ SELECT ▶ UPDATE ▶ SELECT ▶ DELETE ▶ SELECT`

（HTML教材では各項目を `details`/`summary` のアコーディオンで表示）

## 3. 実戦演習（目安）

- 初期データ投入 → SELECT → UPDATE → SELECT → DELETE → SELECT の手順を一通り行う。
