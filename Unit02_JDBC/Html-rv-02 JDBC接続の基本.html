JDBC（Java Database Connectivity）は、Javaプログラムとデータベースを繋ぐ「架け橋」となる標準APIです。DBの種類が変わってもJava側を大きく変えずに済むよう、ドライバが差分を吸収します。

動画で確認すること：Javaアプリ、JDBC、ドライバ、DBの4層の関係。
