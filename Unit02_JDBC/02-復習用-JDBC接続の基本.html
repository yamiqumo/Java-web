図でざっくり復習（JDBC 全体像）

本編テキストの内容を、NotebookLM の図でもう一度なぞりながら復習します。動画を見る前に、下の5枚を上から順に眺めてください。

① JDBC 4ステップの全体像

「接続 → 準備 → 実行 → 解放」という型を、1枚で思い出すための図です。

② DB接続URLの読み方

URL 文字列のどの部分が「ホスト名」「DB名」「ユーザ名」「パスワード」に対応しているかを復習します。

③ executeQuery / executeUpdate の分岐

SELECT 系は executeQuery、INSERT/UPDATE/DELETE は executeUpdate という分岐を再確認してください。

④ ResultSet のカーソルと next()

カーソルが rs.next() で1行ずつ進み、各行の値を getter メソッドで取り出すイメージをつかみます。

⑤ PreparedStatement の威力

プレースホルダ ? に後から値を流し込むことで、同じ型のSQLを安全に何度も実行できることを思い出してください。
