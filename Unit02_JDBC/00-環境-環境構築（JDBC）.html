環境構築：JDBC（MySQL Connector/J）を追加して接続確認

0. ここでやること（最短ルート）

このページは、Java から MySQL に接続できる状態を作るための手順です。

  • mysql-connector-j-8.0.33.jar を入手する（learningBOX）。
  • Eclipse の ビルドパスに外部JARとして追加する。
  • SampleConnection.java を実行し、接続だけ成功することを確認する。

1. JAR をダウンロード（learningBOX）

  1. 以下のリンクを開いて learningBOX にログインする。
  2. 添付から mysql-connector-j-8.0.33.jar をダウンロードする。
  3. ダウンロード先（例：Downloads/）を確認する。
learningBOX（添付DL）

コラム：MySQL公式アーカイブからのDLも可（参考）

learningBOX が開けない場合などは、MySQL公式のアーカイブからも入手できます。

  • https://downloads.mysql.com/archives/c-j/
  • Product Version を 8.0.33。
  • Operating System を Platform Independent。
  • ZIP をDL → 展開 → mysql-connector-j-8.0.33.jar を取り出す。

2. Eclipse に外部JARを追加（ビルドパス）

  1. Javaプロジェクトを右クリック。
  2. ビルド・パス → ビルド・パスの構成。
  3. 上のタブを ライブラリー にする。
  4. 外部JARの追加 から mysql-connector-j-8.0.33.jar を追加 → 適用。

3. 接続確認（SampleConnection を実行）

次の最小プログラムを src 配下に作成して実行します（try-with-resources で接続を自動的に閉じます）。

ファイルは既に用意してあります：Unit02_JDBC/src/SampleConnection.java

4. よくあるエラー（最小）

  • MySQL ドライバが見つからない（ClassNotFoundException）：JAR がビルドパスに入っていない。
  • Access denied：USER/PASS が違う。
  • Unknown database：DB名が違う。
  • Communications link failure：ホスト/ポート、MySQL 起動状態を確認。
