JDBC 基本問題（items・emps）〜ResultSet からデータを取り出して Java で加工する〜

この演習の目的

JDBC の4ステップ（接続→準備→実行→結果取得）を守りながら、ResultSet から値を取り出し、Java 側で集計・加工・出力する練習です。

  • 前半（items）：単一テーブルから値を取り出し、ループ・条件判定・計算を行う。
  • 後半（emps）：複数カラムを組み合わせ、特定条件での抽出・集計に慣れる。
  • すべての問題で try-with-resources を使い、リソースを適切に閉じること。

基本問題 10問（items：商品管理）

目標：ResultSet から値を取り出し、Java で集計・加工する

問題1：全件表示 — 全商品の名前と単価をコンソールに表示せよ。

ヒント

SELECT item_name, unit_price FROM items で全件取得し、while (rs.next()) でループしながら rs.getString / rs.getInt で取り出します。

解答例 
String sql = "SELECT item_name, unit_price FROM items";
try (Connection con = DriverManager.getConnection(URL, USER, PASS);
     PreparedStatement ps = con.prepareStatement(sql);
     ResultSet rs = ps.executeQuery()) {
	while (rs.next()) {
		String name = rs.getString("item_name");
		int price   = rs.getInt("unit_price");
		System.out.println(name + " : " + price + "円");
	}
}

問題4：平均価格の算出 — 全商品の平均価格を計算して表示せよ。

ヒント

合計と件数の両方をループで集め、ループ後に total / count（整数除算）または (double)total / count で平均を求めます。

解答例 
String sql = "SELECT unit_price FROM items";
int total = 0;
int count = 0;
try (Connection con = DriverManager.getConnection(URL, USER, PASS);
     PreparedStatement ps = con.prepareStatement(sql);
     ResultSet rs = ps.executeQuery()) {
	while (rs.next()) {
		total += rs.getInt("unit_price");
		count++;
	}
}
if (count > 0) {
	System.out.println("平均：" + (double) total / count + "円");
}

問題6：最高値の探索 — リストの中で最も高い商品の名前と価格を表示せよ。

ヒント

int maxPrice = 0; String maxName = ""; をループ外に持ち、ループ中に比較して更新します。

解答例 
String sql = "SELECT item_name, unit_price FROM items";
int maxPrice    = 0;
String maxName  = "";
try (Connection con = DriverManager.getConnection(URL, USER, PASS);
     PreparedStatement ps = con.prepareStatement(sql);
     ResultSet rs = ps.executeQuery()) {
	while (rs.next()) {
		int price = rs.getInt("unit_price");
		if (price > maxPrice) {
			maxPrice = price;
			maxName  = rs.getString("item_name");
		}
	}
}
System.out.println("最高値：" + maxName + " " + maxPrice + "円");
